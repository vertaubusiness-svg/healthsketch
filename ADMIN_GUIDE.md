# 헬스스케치 관리자 사용 가이드

---

## 관리자 페이지 접속 방법

웹 브라우저(크롬 권장)를 열고 아래 주소를 입력하세요:

```
https://vertaubusiness-svg.github.io/healthsketch/admin.html
```

> ⚠️ 이 페이지는 검색 엔진에 노출되지 않으며, 위 주소를 직접 입력해야만 접속할 수 있습니다.

---

## 로그인 방법

1. 관리자 페이지에 접속하면 이메일/비밀번호 입력창이 나타납니다.
2. 등록된 관리자 이메일과 비밀번호를 입력하세요.
3. **로그인** 버튼을 클릭합니다.
4. 로그인이 완료되면 관리자 대시보드 화면으로 자동 전환됩니다.

---

## 제품 추가하는 법

1. 로그인 후 왼쪽 **"제품 추가"** 폼을 확인합니다.
2. 각 항목을 입력합니다:
   - **제품명**: 예) 라이프피트니스 케이블머신
   - **카테고리**: 드롭다운에서 해당 항목 선택
     - 플레이트머신 / 유산소 / 오리지널머신 / 프리웨이트 / 기타
   - **설명**: 제품 상태, 특이사항 등 자유롭게 입력
   - **상태**: 드롭다운에서 선택 (기본값: 판매중)
3. **사진 올리기**:
   - "클릭 또는 드래그하여 업로드" 영역을 클릭합니다.
   - 사진 파일을 선택합니다 (JPG, PNG 등, 최대 5장).
   - 선택한 사진 미리보기가 아래에 나타납니다.
   - 잘못 선택한 사진은 미리보기의 × 버튼으로 제거할 수 있습니다.
4. **저장** 버튼을 클릭합니다.
5. 사진이 있는 경우 "업로드 중 1/1…" 메시지가 잠시 표시된 후 저장됩니다.
6. 저장 완료 시 화면 오른쪽 하단에 **"제품이 추가되었습니다."** 알림이 표시됩니다.

---

## 제품 수정하는 법

1. 오른쪽 **"등록 제품 목록"**에서 수정할 제품을 찾습니다.
2. 해당 제품 오른쪽의 **수정** 버튼을 클릭합니다.
3. 왼쪽 폼에 기존 내용이 자동으로 채워집니다.
4. 변경할 내용을 수정합니다.
   - 기존 사진: 미리보기의 × 버튼으로 제거 가능
   - 새 사진 추가: 업로드 영역 클릭 후 선택
5. **저장** 버튼을 클릭합니다.
6. **"제품이 수정되었습니다."** 알림이 표시되면 완료입니다.

> 수정을 취소하려면 폼 상단의 **취소** 버튼을 클릭하세요.

---

## 제품 삭제하는 법

1. 오른쪽 **"등록 제품 목록"**에서 삭제할 제품을 찾습니다.
2. 해당 제품 오른쪽의 **삭제** 버튼을 클릭합니다.
3. "정말 삭제하시겠습니까?" 확인 창이 나타납니다.
4. **확인**을 클릭하면 즉시 삭제됩니다.

> ⚠️ **삭제한 제품은 복구할 수 없습니다.** 신중하게 진행하세요.
>
> 판매가 완료된 제품은 삭제 대신 상태를 **"판매완료"**로 변경하는 것을 권장합니다.

---

## 제품 상태 바꾸는 법 (판매중 → 판매완료 등)

1. 오른쪽 목록에서 해당 제품의 **수정** 버튼을 클릭합니다.
2. 왼쪽 폼에서 **"상태"** 드롭다운을 찾습니다.
3. 원하는 상태로 변경합니다:
   - **판매중**: 판매 가능한 제품 (기본값)
   - **예약중**: 구매 예약이 된 제품
   - **판매완료**: 판매가 끝난 제품 (목록에서 흐리게 표시됨)
4. **저장** 버튼을 클릭합니다.

---

## 로그아웃 방법

관리자 대시보드 상단 오른쪽의 **로그아웃** 버튼을 클릭합니다.

---

## 자주 묻는 질문 (FAQ)

### ❓ 로그인이 안 돼요

- 이메일과 비밀번호를 정확히 입력했는지 확인하세요 (대소문자 구분).
- 비밀번호를 잊으셨다면 담당자(개발자)에게 문의하세요.
- 여러 번 틀리면 잠시 잠금이 걸릴 수 있습니다. 몇 분 후 다시 시도하세요.

### ❓ 사진이 안 올라가요

- 인터넷 연결 상태를 확인하세요.
- 파일이 JPG, PNG, WEBP 형식인지 확인하세요.
- 한 번에 최대 5장까지만 업로드할 수 있습니다.
- 파일 크기가 너무 크면 시간이 오래 걸릴 수 있습니다. 잠시 기다려 보세요.
- 그래도 안 되면 사진 없이 저장 후 나중에 수정으로 추가해 보세요.

### ❓ 저장했는데 목록에 안 보여요

- 페이지를 새로고침(F5) 해보세요.
- 인터넷이 불안정한 경우 잠시 후 다시 확인하세요.

### ❓ 제품을 추가했는데 홈페이지에 바로 안 보여요

- 보통 수 초 내에 자동으로 반영됩니다.
- 홈페이지(products.html)를 새로고침(F5) 해보세요.

### ❓ 사진을 잘못 올렸어요, 어떻게 바꾸나요?

- 해당 제품의 **수정** 버튼을 클릭합니다.
- 기존 사진 미리보기에서 × 버튼으로 잘못된 사진을 제거합니다.
- 새 사진을 업로드 후 저장합니다.

### ❓ 판매완료 제품이 목록에 계속 보이는데 지워야 하나요?

- 꼭 삭제하지 않아도 됩니다.
- 판매완료 제품은 홈페이지에서 흐리게 표시되어 구분되므로, 기록 보존을 위해 그대로 두셔도 됩니다.
